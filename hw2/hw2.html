Task 3

  1. What is the type of variable Passengers? (Categorical nominal/ordinal or Numerical discrete/continuous?) What happens if we were to make a histogram for Passengers? What is the problem with this graph? The variable Passengers is categorical nominal because each value represents the number of passengers a car model can seat. Since histograms show the distribution of numerical data, either discrete or continuous, the categorical nominal variable is not an appropriate choice because there is no numberical relationship between these categories. Instead, a bar graph or pie chart would be a better option to visualize the frequency.

  2. Correct the problem in part (a) by creating a more appropriate visualization. Make sure to label your axes, place a title, choose a custom color as in the previous graphs. (Bonus: use factor() with levels option to display x axis values with no observations on the plot.)

library(MASS)
